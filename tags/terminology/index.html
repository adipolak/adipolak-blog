Machine Learning in Production - Concepts you should know

418 Words

Are you interested in learning about the Machine Learning side of data? Hurry 🎉 , you have reached the right place to start learning about it. Here is a list of concepts for you to get started: ML Algorithm ML algorithm is a procedure that runs on data and produces a machine learning model. Some of the popular ones are Decision trees, Naive Bayes, and Linear Regression. ML Model ML model is the ML algorithm process outcome; It often contains a statistical representation of the data ingested into the algorithm.
